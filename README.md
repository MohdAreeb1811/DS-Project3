# 💻 Laptop Price Prediction & Analysis

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** and **Machine Learning** on a laptop dataset to:

* Analyze factors affecting laptop prices
* Classify laptops into price categories
* Predict exact laptop prices using regression

---

## 🚀 Features

* 📊 Data Cleaning & Preprocessing
* 📈 Exploratory Data Analysis (EDA)
* 🧠 Feature Engineering
* 🤖 Machine Learning Models:

  * Classification (Price Category)
  * Regression (Price Prediction)

---

## 🛠️ Tech Stack

* Python 🐍
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn

## 📌 Results

* ✅ Classification Accuracy: ~82%
* ✅ Regression R² Score: ~0.73

---

## 📈 Key Insights

* RAM has strong impact on price 💾
* SSD laptops are generally more expensive ⚡
* High-end brands (Apple, Razer) have higher price ranges 💰
* Screen resolution also influences pricing 🖥️
